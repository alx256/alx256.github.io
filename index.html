LLMVis

LLMVis is a Python module that uses data visualization and explainable AI to help developers better understand the results of their generative AI applications. The tool was developed using Python, JavaScript, HTML and CSS and it was done for my final year undergraduate project.

Sonny

Sonny is a cross platform virtual assistant for elderly users, helping them with day-to-day activities and staying connected with their loved ones to aid with the problem of loneliness that is common amongst the elderly. It was developed for my second year group project using the Flutter framework and the Dart programming language.

Echo Launcher

Echo Launcher is a custom Android launcher written in Java using the Android Development Environment. As well as providing users with a fast and modern home screen experience, it is also built for students, featuring key productivity features such as "Study Mode", which when enabled allows students to focus on studying, free from distractions from their phone.

Visuomotor Game

For my Extended Project Qualification, I produced a game for improving the visuomotor control of its players. Research has shown that action video games have been linked to an improvement in visuomotor skills crucial for daily activities such as driving. This game incorporates key attributes from games in which these effects have been observed to positively impact the players. I wrote it in the Unity game engine using C#

Joystick

Joystick is a build system for building software using a multi-threaded approach in C++. It is fast, interactive and easy to use, and can be used for building a wide range of software projects.

2D Batch Renderer

I have written a 2D renderer in C++ using OpenGL for writing high-performing 2D games, using the concept of batching to draw many items in a single draw call.
